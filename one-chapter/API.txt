网际套接字地址结构中IP地址和端口号这两个成员必须使用特定格式

通用套接字地址结构：
	 struct sockaddr 
	 {
		sa_family_t sa_family;
	    char        sa_data[14];
	 };
	 
1. 每当一个套接字函数需要一个指向某个套接字地址结构的指针的时候，这个指针必须强制类型转换成通用套接字地址结构的指针

2. 当服务器主机有多个网口的时候，将服务器地址设置为INADDR_ANY，服务器进程就可以在任意网络接口上接受客户程序

3. 通常情况下，服务器进程在调用accept中被投入睡眠，等待客户连接的到来并被内核接受。

4. 服务器通过调用close关闭与客户的连接。该调用引发正常的TCP连接终止序列：每个发送序列上发送一个FIN,每个FIN又由各自对端确认。
